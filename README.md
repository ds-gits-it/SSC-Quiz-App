# Welcome to your SSC Quiz App 👋

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Get started

1. Setup Environment by following guide at https://docs.expo.dev/get-started/set-up-your-environment
2. Install dependencies
   ```bash
   npm install
   ```
3. Run on android device or ios device with below commands respectively
   ```bash
   npm run android
   ```
   ```
   npm run ios
   ```
